# Converted Output Samples

This folder contains small downscaled PNG previews made from converted 16-bit RGB TIFF outputs.

The samples are included so GitHub readers can quickly see the expected visual result without downloading full-size TIFF files.

- `000_IMG_0100_sample.png`
- `001_IMG_0280_sample.png`
- `flight_IMG_0300_sample.png`

These previews are for visual reference only, not quantitative analysis.
